Bem-vindo ao sistema!

Objetivo

O presente sistema foi desenvolvido para a avaliação prática da primeira etapa da disciplina de Programação WEB, ministrada pelo professor Jorge Bavaresco no semestre 2022/2.

O objetivo foi a implementação de uma camada de modelo com no mínimo 5 classes persistentes utilizando JPA e com validações em todos os os seus atributos, com pelo menos uma das classes possuindo um relacionamento de muitos para um.

A camada web, por sua vez, foi desenvolvida com Java Serves Faces e proporciona manutenções CRUD para pelo menos 3 classes, sendo uma delas com relacionamento muitos para um, e é hospedada em servidor Payara.

Funcionamento

O sistema representa a administração de cadastros de uma loja de varejo chamada Lumi Enxovais que mantém registros acerca do seu negócio (vendas, clientes, produtos e fornecedores).

Utilize o menu ao topo para navegar entre as categorias que deseja visualizar e realizar CRUD.
